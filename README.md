# Planar Sketch Linkage

Interactive planar linkage sketcher with constraints + simulation.

## Run
```bash
python run.py
```

## Features
- Points / Lengths / Angles editing
- Constraints: length, angle, coincide, point-on-line, rigid bodies
- Parameter expressions on coordinates/lengths/angles
- Simulation sweep (PBD + optional SciPy backend)
